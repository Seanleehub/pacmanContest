# pacmanContest

Created a competitive agent for Pac Man Capture the Flag game from the UC Berkeley Intro to AI assignment. 
It is a multi-player game with two agents on each team. The goal is to get food in the opponents’ territory and bring the food back to your home side, while 
defending the opponents which are trying to do the same.

Done in Python incorperating search algorithm's, Multi-Agent Search and Reinforcement Learning.

Report avaiable in pdf format.
